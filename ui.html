Interpolation results for different microbiome species

Instructions:
This Rshiny APP shows the interpolation results using certain deep learning methods. The information shown in solid points are used to train the model. However, the rest information was not used for the training process. The lines are the time series interpolated using the deep learning method.
The data belongs to multivariate time series datasets. For different subjects, there are 5 different microbiome species following different time trends. We can change the microbiome species to show it. And we can select different subjects to see the microbiome time series in different people.
